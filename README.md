# RIRD Desktop Releases

Download the latest RIRD Desktop installer from the [Releases](https://github.com/0xb00c/rird-releases/releases) page.

## Installation

1. Download the installer for your platform
2. Run the installer
3. Launch RIRD and enter your license key

## System Requirements

- **Windows**: Windows 10/11 (64-bit)
- **macOS**: macOS 11.0+ (Apple Silicon)
- **Linux**: Ubuntu 20.04+ or equivalent

[Learn more at rird.ai](https://rird.ai)
